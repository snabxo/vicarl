## VICARL

Vicar Ledger is an embeddable, modular, lightweight ledger as a library that can act as a tamper evident storage layer inside consumer/embedded apps and works well on mobile platforms with limited IO, battery and intermittent connectivity. Vicarl supports lightweight P2P replication and exchange of ledgered data and can integrate cleanly into existing systems as a storage, audit or replication component.

Vicarl is an experimental system to test and explore an alternate lightweight and flexible path to implementing auditing in a microservice architectured environment where data protection, privacy and security is a priority and it can be thought of as an append-only, cryptographically verifiable log with optional state/indexing and P2P sync.
